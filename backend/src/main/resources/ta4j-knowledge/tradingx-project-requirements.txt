TradingX Project Strategy Code Requirements
============================================

== Required Code Structure ==

Generated strategy code MUST follow this structure:

package com.tradingx.strategy;

import org.ta4j.core.BarSeries;
import org.ta4j.core.BaseStrategy;
import org.ta4j.core.Rule;
import org.ta4j.core.Strategy;
// ... other ta4j v0.22.6 imports

public class MyStrategy {
    public static Strategy buildStrategy(BarSeries series) {
        // Create indicators
        // Build entry and exit rules
        // Return strategy
        return new BaseStrategy("MyStrategy", entryRule, exitRule);
    }
}

== Key Requirements ==

1. Package name MUST be: com.tradingx.strategy
2. Class MUST be public
3. MUST have a static method: public static Strategy buildStrategy(BarSeries series)
4. MUST return a BaseStrategy instance
5. ONLY import from ta4j v0.22.6 - do NOT use APIs from other versions

== Common Mistakes to Avoid ==

WRONG (old API): import org.ta4j.core.indicators.SMAIndicator;
CORRECT (v0.22.6): import org.ta4j.core.indicators.averages.SMAIndicator;

WRONG (old API): import org.ta4j.core.indicators.ADXIndicator;
CORRECT (v0.22.6): import org.ta4j.core.indicators.adx.ADXIndicator;

WRONG (old API): import org.ta4j.core.indicators.BollingerBandsMiddleIndicator;
CORRECT (v0.22.6): import org.ta4j.core.indicators.bollinger.BollingerBandsMiddleIndicator;

WRONG (old API): import org.ta4j.core.indicators.helpers.TRIndicator;
CORRECT (v0.22.6): import org.ta4j.core.indicators.helpers.TRIndicator; (same path)

WRONG (old API): new BaseStrategy(entryRule, exitRule);
CORRECT (v0.22.6): new BaseStrategy("StrategyName", entryRule, exitRule);

== v0.22.6 Package Changes from Older Versions ==

- SMAIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.averages
- EMAIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.averages
- WMAIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.averages
- LWMAIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.averages
- ADXIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.adx
- PlusDIIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.adx
- MinusDIIndicator moved from org.ta4j.core.indicators to org.ta4j.core.indicators.adx
- BollingerBandsMiddleIndicator moved to org.ta4j.core.indicators.bollinger
- BollingerBandsUpperIndicator moved to org.ta4j.core.indicators.bollinger
- BollingerBandsLowerIndicator moved to org.ta4j.core.indicators.bollinger
- Ichimoku indicators moved to org.ta4j.core.indicators.ichimoku
- Volume indicators moved to org.ta4j.core.indicators.volume
- Candlestick indicators moved to org.ta4j.core.indicators.candles
- Statistical indicators moved to org.ta4j.core.indicators.statistics
- BaseStrategy now requires a name as the first parameter
- BarSeriesManager is now in org.ta4j.core.backtest package
- DecimalNumFactory is used instead of DoubleNumFactory
